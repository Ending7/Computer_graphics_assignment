[m]: 도형의 모드: LINE/FILL
[p]: 경로 출력하기: on/off
[+]: 날라오는 속도 증가
[-]: 날라오는 속도 감소
[q]: 프로그램 종료

------------모든 [기본]구현 완료하였으며 슬라이스 어려운 버전으로 구현하였습니다.------------
[화면의 좌측, 또는 우측 가장자리의 임의의 위치에서 도형이 화면의 반대편으로 날라온다]
   -날아오는 경로: 사선으로 설정한다.
   -도형의 종류: 삼각형 이상의 면을 가지고 있는 2 종류 이상의 도형(최소 삼각형과 사각형)

[아래에는 바구니가 좌우로 움직이고 있다]
   -화면 아래에 바구니 그리기
   -바구니 좌우로 이동

[마우스를 클릭, 드래그하여 도형을 슬라이스한다]
   -마우스를 클릭, 드래그하여 빨간색 선을 그린다
   -드래그하여 도형을 슬라이스한다.
   -조각이 떨어질 때는 중력이 적용되어 떨어진다.

[슬라이스 하지 못하면 반대편으로 사라진다]

[종료 명령어가 입력될 때까지 계속 도형이 날라온다]

[키보드 명령어] (완료)
-도형의 모드:LINE/FILL (완료) //키보드 m
-경로 출력하기:on/off (완료) //키보드 p
-날라오는 속도 증가(완료) //키보드 +
-날라오는 속도 감소(완료) //키보드 -
-프로그램 종료 (완료) // 키보드 q

------------보너스 문제 구현------------
1.슬라이스를 할 때 자른 모양대로 자유롭게 도형이 두개로 나뉜다. 자유 슬라이스(완료)
2.도형이 날라올 때 사선도 뿐만 아니라(기본) 곡선도 섞여 나오도록(보너스) 추가함. 배지어 3차공식을 이용함.(완료)
3.p를 누르면 사선으로 날라오는 도형 뿐만 아니라(기본) 포물선으로 날라오는 도형도(보너스) ""경로""가 그려지도록 함.
4.도형 회전하며 날라오기
5.프로그램 시작했을 때 날라오는 경로 직접 정하기
