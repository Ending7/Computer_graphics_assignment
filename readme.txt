------------보너스 문제 구현------------
1.슬라이스를 할 때 자른 모양대로 자유롭게 도형이 두개로 나뉜다: 자유 슬라이스(보너스 문제 풀이: 어려운 버전) //모양은 자유롭게 슬라이스 할 수 있지만 횟수는 한번만 슬라이스 가능합니다.

2.도형의 이동: 사선으로 이동하는 도형도 생성되고(기본 문제 풀이) + 곡선으로 이동하는 도형도 생성되도록 추가함(보너스 문제 풀이) //사선으로 날아오는 것은 컨트롤포인트 2개, 곡선은 컨트롤포인트 4개를 이용하였습니다.

3.p를 누르면 사선으로 이동하는 도형의 경로가 화면에 그려질 뿐만 아니라 (기본 문제 풀이) + 곡선으로 이동하는 도형의 경로도 화면에 그려짐(보너스 문제 풀이) 
/***곡선으로 이동하는 도형의 경로를 출력하는 방법은 사선으로 이동하는 것과 완전히 다릅니다.***/

4.도형 회전하며 날라오기: 도형이 날라올 때 평범하게 날아오지 않고 빙글빙글 돌면서 날라옴. object moves while spinning.(보너스 문제 풀이) //cos, sin을 이용한 회전을 사용하였습니다.
